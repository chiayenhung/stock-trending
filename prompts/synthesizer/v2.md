# Synthesis task

Synthesize the analyst outputs into research-only trend signals. Every signal
must trace to claim_ids and include an assessment, research horizon, thesis,
monitoring triggers, and exactly the configured 5-session, 21-session, and
63-session outlooks. Preserve the source analyst's 1-to-10 assessment-strength
score as `signal_strength_score`; this is not an expected-return score. Preserve
the outlook probability basis as
`model_estimate_uncalibrated` and keep every outlook's supporting claim IDs
within the signal's evidence claim IDs. Set `research_only` to true and
`validation_status` to `pending`; deterministic code owns the final validation
status. Do not call an outlook a historical win rate or profit forecast. Do not
produce transaction instructions, price levels, or quantities. Return the
requested JSON schema.
